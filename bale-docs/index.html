مستندات بازوی بله

نحوه استفاده

هر ماژول یک فایل HTML مستقل است که می‌تواند به صورت جداگانه بارگذاری شود. این ساختار مزایای زیر را دارد:

  • بارگذاری سریع‌تر صفحات
  • خوانایی بهتر برای LLMها
  • نگهداری آسان‌تر
  • قابلیت جستجوی بهتر
