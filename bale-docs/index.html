مستندات API بازوی بله

راهنمای جامع توسعه بازو (بات) و مینی‌اپ برای پلتفرم بله

خوش آمدید

این مستندات رسمی API بازوی بله است. API بازوی بله بر پایهٔ API بات تلگرام و با اعمال تغییراتی جزئی طراحی شده است؛ بنابراین می‌توانید از کتابخانه‌های متعددی که برای توسعهٔ بات تلگرام در زبان‌های مختلف ارائه شده‌اند، برای ساخت بازو در بله نیز استفاده کنید.

این API از ساختاری ساده و رایج مبتنی بر HTTP بهره می‌برد و به‌راحتی قابل استفاده است.

ساختار مستندات

این مستندات به صورت ماژولار طراحی شده است تا خواندن و درک آن برای انسان و مدل‌های زبانی بزرگ (LLM) آسان باشد. هر بخش به صورت جداگانه در فایل‌های HTML مجزا قرار دارد.

بخش‌های اصلی:

  • شروع سریع: شامل ساخت بازو، ایجاد درخواست و دریافت آپدیت‌ها
  • متدهای دریافت آپدیت: توضیحات کامل getUpdates، setWebhook و سایر متدها
  • انواع داده‌ها: معرفی تمام انواع داده‌های مورد استفاده در API
  • ارسال فایل‌ها: راهنمای کامل آپلود و ارسال فایل
  • مینی‌اپ‌ها: مستندات توسعه مینی‌اپ برای بله

لینک‌های مفید

  • بازوی پدر (@botfather) - برای ساخت بازوی جدید
  • API Endpoint - نقطه دسترسی API

نحوه استفاده از این مستندات

برای شروع، پیشنهاد می‌کنیم از بخش ساخت بازو آغاز کنید. سپس با نحوه ایجاد درخواست آشنا شوید و در نهایت روش‌های دریافت آپدیت‌ها را یاد بگیرید.

هر ماژول به صورت مستقل قابل مطالعه است و شامل تمام اطلاعات لازم برای آن موضوع خاص می‌باشد.
