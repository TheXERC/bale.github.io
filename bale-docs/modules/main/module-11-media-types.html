انواع رسانه‌ای و فایل

ساختارهای داده‌ای مربوط به فایل‌های رسانه‌ای در بله

PhotoSize

این شیء نمایندهٔ یک اندازه از عکس است.

فیلد نوع توضیحات
file_id String شناسه این فایل که می‌توان آن را جهت دانلود یا استفاده مجدد به کار برد.
file_unique_id String شناسه منحصربه‌فرد این فایل که در طول زمان و برای بازوهای مختلف یکسان است.
width Integer عرض عکس
height Integer ارتفاع عکس
file_size Integer اختیاری. حجم فایل با واحد بایت
