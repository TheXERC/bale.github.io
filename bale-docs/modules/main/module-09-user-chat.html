انواع کاربر و گفتگو

تعریف ساختارهای داده‌ای مربوط به کاربران و گفتگوها در API بله

User

این شیء نمایندهٔ یک کاربر در بله است.

فیلد نوع توضیحات
id Integer شناسه یکتای کاربر. می‌تواند بیشتر از ۲۳۱ باشد و برخی زبان‌های برنامه‌نویسی ممکن است در تفسیر آن با مشکل روبه‌رو شوند.
is_bot Boolean آیا کاربر یک بازو است؟
first_name String نام کوچک کاربر
last_name String اختیاری. نام خانوادگی کاربر
username String اختیاری. نام کاربری کاربر
language_code String اختیاری. کد زبان ترجیحی کاربر
is_premium Boolean اختیاری. آیا کاربر اشتراک پریمیوم دارد؟
added_to_attachment_menu Boolean اختیاری. آیا این بازو به منوی پیوست‌ها اضافه شده است؟

Chat

این شیء نمایندهٔ یک گفتگو (چت) در بله است.

فیلد نوع توضیحات
id Integer شناسه یکتای گفتگو. می‌تواند بیشتر از ۲۳۱ باشد.
type String نوع گفتگو که می‌تواند یکی از موارد private، group یا channel باشد.
title String اختیاری. عنوان گفتگو برای گروه‌ها یا کانال‌ها
username String اختیاری. نام کاربری برای گفتگوهای خصوصی یا کانال‌ها
first_name String اختیاری. نام کوچک طرف مقابل در گفتگوی خصوصی
last_name String اختیاری. نام خانوادگی طرف مقابل در گفتگوی خصوصی

ChatFullInfo

اطلاعات کامل‌تر دربارهٔ یک گفتگو.

فیلد نوع توضیحات
id Integer شناسه یکتای گفتگو
type String نوع گفتگو
title String اختیاری. عنوان گفتگو
username String اختیاری. نام کاربری
first_name String اختیاری. نام کوچک
last_name String اختیاری. نام خانوادگی
photo ChatPhoto اختیاری. تصویر گفتگو
bio String اختیاری. متن بخش «دربارهٔ من» کاربر
description String اختیاری. توضیحات گروه یا کانال
invite_link String اختیاری. پیوند دعوت
linked_chat_id Integer اختیاری. شناسه گفتگوی مرتبط
قبلی: getWebhookInfo بعدی: Message Types
