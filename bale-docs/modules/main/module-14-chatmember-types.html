انواع اعضای گفتگو

ساختارهای داده‌ای مربوط به اعضای گروه‌ها و کانال‌ها

ChatMember

این شیء نمایندهٔ یک عضو در گفتگو (گروه یا کانال) است. نوع عضو مشخص می‌کند که کدام یک از انواع زیر استفاده شود.

انواع ChatMember:

  • ChatMemberOwner - مالک گفتگو
  • ChatMemberAdministrator - مدیر گفتگو
  • ChatMemberMember - عضو عادی
  • ChatMemberRestricted - عضو محدود شده
  • ChatMemberLeft - عضوی که ترک کرده (در این مستندات پوشش داده نشده)
  • ChatMemberBanned - عضوی که بن شده (در این مستندات پوشش داده نشده)

ChatMemberOwner

نمایندهٔ مالک یک گفتگو.

فیلد نوع توضیحات
status String همیشه creator است.
user User اطلاعات کاربر مالک
is_anonymous Boolean آیا مالک به صورت ناشناس نمایش داده می‌شود؟
custom_title String اختیاری. عنوان سفارشی مالک

ChatMemberAdministrator

نمایندهٔ یک مدیر در گفتگو.

فیلد نوع توضیحات
status String همیشه administrator است.
user User اطلاعات کاربر مدیر
can_be_edited Boolean آیا می‌توان وضعیت این مدیر را تغییر داد؟
is_anonymous Boolean آیا مدیر به صورت ناشناس نمایش داده می‌شود؟
can_manage_chat Boolean آیا مدیر می‌تواند گفتگو را مدیریت کند؟
can_delete_messages Boolean آیا مدیر می‌تواند پیام‌ها را حذف کند؟
can_manage_video_chats Boolean آیا مدیر می‌تواند چت‌های ویدیویی را مدیریت کند؟
can_restrict_members Boolean آیا مدیر می‌تواند اعضا را محدود کند؟
can_promote_members Boolean آیا مدیر می‌تواند اعضا را ارتقا دهد؟
can_change_info Boolean آیا مدیر می‌تواند اطلاعات گفتگو را تغییر دهد؟
can_invite_users Boolean آیا مدیر می‌تواند کاربران را دعوت کند؟
can_post_messages Boolean اختیاری. آیا مدیر می‌تواند در کانال پست بگذارد؟ (فقط کانال)
can_edit_messages Boolean اختیاری. آیا مدیر می‌تواند پیام‌ها را ویرایش کند؟ (فقط کانال)
can_pin_messages Boolean اختیاری. آیا مدیر می‌تواند پیام‌ها را سنجاق کند؟
can_manage_topics Boolean آیا مدیر می‌تواند موضوعات را مدیریت کند؟
custom_title String اختیاری. عنوان سفارشی مدیر

ChatMemberMember

نمایندهٔ یک عضو عادی در گفتگو.

فیلد نوع توضیحات
status String همیشه member است.
user User اطلاعات کاربر عضو

ChatMemberRestricted

نمایندهٔ یک عضو محدود شده در گفتگو.

فیلد نوع توضیحات
status String همیشه restricted است.
user User اطلاعات کاربر محدود شده
is_member Boolean آیا کاربر عضو فعلی گفتگو است؟
can_send_messages Boolean آیا کاربر می‌تواند پیام ارسال کند؟
can_send_audios Boolean آیا کاربر می‌تواند فایل صوتی ارسال کند؟
can_send_documents Boolean آیا کاربر می‌تواند فایل سند ارسال کند؟
can_send_photos Boolean آیا کاربر می‌تواند عکس ارسال کند؟
can_send_videos Boolean آیا کاربر می‌تواند ویدیو ارسال کند؟
can_send_video_notes Boolean آیا کاربر می‌تواند ویدیو نوت ارسال کند؟
can_send_voice_notes Boolean آیا کاربر می‌تواند پیام صوتی ارسال کند؟
can_send_polls Boolean آیا کاربر می‌تواند نظرسنجی ارسال کند؟
can_send_other_messages Boolean آیا کاربر می‌تواند سایر پیام‌ها را ارسال کند؟
can_add_web_page_previews Boolean آیا کاربر می‌تواند پیش‌نمایش وب‌صفحه اضافه کند؟
can_change_info Boolean آیا کاربر می‌تواند اطلاعات گفتگو را تغییر دهد؟
can_invite_users Boolean آیا کاربر می‌تواند کاربران را دعوت کند؟
can_pin_messages Boolean آیا کاربر می‌تواند پیام‌ها را سنجاق کند؟
until_date Integer زمان پایان محدودیت به صورت یونیکس

ChatPhoto

این شیء نمایندهٔ تصویر یک گفتگو است.

فیلد نوع توضیحات
small_file_id String شناسه فایل تصویر کوچک (۱۶۰x۱۶۰)
small_file_unique_id String شناسه منحصربه‌فرد تصویر کوچک
big_file_id String شناسه فایل تصویر بزرگ (۶۴۰x۶۴۰)
big_file_unique_id String شناسه منحصربه‌فرد تصویر بزرگ
