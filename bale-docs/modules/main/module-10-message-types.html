فیلد نوع توضیحات
message_id Integer شناسه یکتای پیام

MessageEntity

این شیء نمایندهٔ یک موجودیت ویژه در متن پیام است (مثل لینک، منشن، هشتگ و...).

فیلد نوع توضیحات
type String نوع موجودیت. می‌تواند یکی از موارد زیر باشد:
  • mention - منشن کاربر (@username)
  • hashtag - هشتگ (#hashtag)
  • bot_command - دستور بازو (/command)
  • url - لینک URL
  • email - آدرس ایمیل
  • phone_number - شماره تلفن
  • bold - متن ضخیم
  • italic - متن کج
  • underline - متن زیرخط‌دار
  • strikethrough - متن خط‌خورده
  • code - کد تک‌خطی
  • pre - کد چندخطی
  • text_link - لینک متنی
  • text_mention - منشن متنی
offset Integer موقعیت شروع موجودیت در متن پیام
length Integer طول موجودیت
url String اختیاری. URL برای انواع text_link
user User اختیاری. کاربر برای انواع text_mention
language String اختیاری. زبان برنامه‌نویسی برای نوع pre
