setWebhook

از این متد برای تعیین یک URL وب‌هوک خروجی استفاده می‌شود که از طریق آن Update ها دریافت می‌شوند. زمانی که یک Update برای بازو وجود دارد، یک درخواست HTTPS POST به URL مشخص‌شده ارسال خواهد شد که حاوی آپدیت JSON-serialized است.

خروجی این متد در صورت اجرای موفقیت‌آمیز True است.

POST https://tapi.bale.ai/bot<token>/setWebhook

پارامترها

پارامتر نوع الزام توضیحات
url String الزامی آدرس HTTPS URL ای که Update ها باید به آن ارسال شوند. برای حذف یکپارچه‌سازی (غیرفعال کردن) وب‌هوک، یک رشته خالی را ارسال کنید.

نکته: درگاه‌هایی (ports) که در حال حاضر برای وب‌هوک پشتیبانی می‌شوند، عبارتند از: 443, 88

مثال درخواست

POST https://tapi.bale.ai/bot123456789:abcdIuZmK5qNEm2A1BhUaAg7MPJv1O9KCcBQB2ro/setWebhook
Content-Type: application/json

{
  "url": "https://example.com/webhook"
}

مثال پاسخ موفق

{
  "ok": true,
  "result": true,
  "description": "Webhook was set"
}

حذف وب‌هوک

برای غیرفعال کردن وب‌هوک و بازگشت به حالت long polling، یک رشته خالی به عنوان URL ارسال کنید:

POST https://tapi.bale.ai/bot123456789:abcdIuZmK5qNEm2A1BhUaAg7MPJv1O9KCcBQB2ro/setWebhook
Content-Type: application/json

{
  "url": ""
}

الگوی پیاده‌سازی وب‌هوک

// مثال ساده با Express.js
const express = require('express');
const app = express();

app.use(express.json());

app.post('/webhook', (req, res) => {
  const update = req.body;
  
  // پردازش آپدیت
  if (update.message) {
    handleMessage(update.message);
  }
  
  // پاسخ موفقیت‌آمیز
  res.status(200).send('OK');
});

function handleMessage(message) {
  const chatId = message.chat.id;
  const text = message.text;
  
  // پاسخ به کاربر
  fetch(`https://tapi.bale.ai/bot${TOKEN}/sendMessage`, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      chat_id: chatId,
      text: `شما گفتید: ${text}`
    })
  });
}

app.listen(3000);

نکات مهم

  • URL وب‌هوک باید حتماً HTTPS باشد
  • سرور شما باید گواهی SSL معتبر داشته باشد
  • پورت‌های پشتیبانی شده: 443 و 88
  • سرور شما باید بتواند درخواست‌های POST را دریافت و پردازش کند
  • همیشه پاسخ موفقیت‌آمیز (کد 200) به سرور بله برگردانید
