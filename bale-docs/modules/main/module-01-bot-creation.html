ساخت بازو

برای ساخت یک بازو در بله، باید مراحل زیر را دنبال کنید:

دریافت توکن از @botfather

برای شروع کار با بازوها، ابتدا باید یک توکن منحصر به فرد از @botfather دریافت کنید. این توکن برای احراز هویت بازوی شما استفاده می‌شود.

مراحل دریافت توکن:

  1. وارد گفتگو با @botfather شوید
  2. دستور /newbot را ارسال کنید
  3. یک نام برای بازوی خود انتخاب کنید
  4. یک نام کاربری (username) برای بازوی خود انتخاب کنید که باید با کلمه bot پایان یابد
  5. توکن دریافتی را ذخیره کنید

نمونه توکن:

1234567890:ABCdefGHIjklMNOpqrsTUVwxyz

هشدار مهم: توکن خود را با کسی به اشتراک نگذارید و آن را در کدهای خود به صورت hard-coded قرار ندهید.

ساختار کلی API

API بازوی بله بر اساس پروتکل HTTPS کار می‌کند. تمام درخواست‌ها باید به آدرس زیر ارسال شوند:

https://tapi.bale.ai/bot<TOKEN>/<METHOD>

که در آن:

  • <TOKEN>: توکن دریافتی از @botfather
  • <METHOD>: نام متدی که می‌خواهید فراخوانی کنید (مثلاً sendMessage)

مثال ارسال درخواست

curl -X POST https://tapi.bale.ai/bot1234567890:ABCdefGHIjklMNOpqrsTUVwxyz/sendMessage \
  -d "chat_id=123456789" \
  -d "text=سلام!"

پارامترهای عمومی

بیشتر متدهای API از پارامترهای زیر پشتیبانی می‌کنند:

پارامتر نوع الزامی توضیحات
chat_id String یا Integer بله شناسه منحصر به فرد گفتگو هدف یا نام کاربری کانال هدف (با فرمت @channelusername)
text String بستگی دارد متن پیام بین ۱ تا ۴۰۹۶ کاراکتر

خطاهای رایج

  • 401 Unauthorized: توکن نامعتبر است
  • 400 Bad Request: پارامترهای درخواست ناقص یا نامعتبر هستند
  • 429 Too Many Requests: تعداد درخواست‌ها بیش از حد مجاز است
