ResponseParameters

این شیء نمایندهٔ پارامترهای پاسخ خطا است که اطلاعات بیشتری دربارهٔ دلیل خطا ارائه می‌دهد.

فیلد نوع توضیحات
migrate_to_chat_id Integer اختیاری. اگر گروه به سوپرگروه ارتقا یافته باشد، شناسه سوپرگروه جدید را نشان می‌دهد.
retry_after Integer اختیاری. تعداد ثانیه‌هایی که باید قبل از ارسال درخواست مجدد صبر کنید (در صورت محدود شدن نرخ).

مثال پاسخ خطا:

{
  "ok": false,
  "error_code": 429,
  "description": "Too Many Requests: retry after 5",
  "parameters": {
    "retry_after": 5
  }
}

InputFile

این نوع نمایندهٔ یک فایل برای آپلود است. InputFile می‌تواند به یکی از روش‌های زیر باشد:

  • رشته (String):
    • اگر رشته با attach:// شروع شود، فایل از طریق multipart/form-data آپلود می‌شود.
    • اگر رشته یک file_id معتبر باشد، فایل قبلاً آپلود شده استفاده می‌شود.
    • اگر رشته یک URL HTTP/HTTPS باشد، فایل از آن آدرس دانلود می‌شود.
  • آپلود مستقیم:
    • فایل به صورت multipart/form-data همراه با درخواست ارسال می‌شود.

نکات مهم:

  • حداکثر حجم فایل برای عکس: ۱۰ مگابایت
  • حداکثر حجم فایل برای ویدیو: ۵۰ مگابایت
  • حداکثر حجم فایل برای سایر فایل‌ها: ۵۰ مگابایت
  • برای استفاده از file_id، فایل باید قبلاً توسط همان بازو آپلود شده باشد.

InputMedia

این نوع نمایندهٔ یک فایل رسانه‌ای عمومی برای آپلود است. انواع مختلف InputMedia عبارتند از:

انواع InputMedia:

  • InputMediaPhoto - عکس
  • InputMediaVideo - ویدیو
  • InputMediaAnimation - انیمیشن (GIF)
  • InputMediaAudio - فایل صوتی
  • InputMediaDocument - فایل سند

InputMediaPhoto

فیلد نوع توضیحات
type String همیشه photo
media String شناسه فایل یا URL
caption String اختیاری. توضیحات عکس (حداکثر ۱۰۲۴ کاراکتر)
parse_mode String اختیاری. حالت قالب‌بندی متن (HTML یا MarkdownV2)

InputMediaVideo

فیلد نوع توضیحات
type String همیشه video
media String شناسه فایل یا URL
thumbnail InputFile اختیاری. تصویر بندانگشتی
caption String اختیاری. توضیحات ویدیو
parse_mode String اختیاری. حالت قالب‌بندی متن
width Integer اختیاری. عرض ویدیو
height Integer اختیاری. ارتفاع ویدیو
duration Integer اختیاری. مدت زمان ویدیو به ثانیه
supports_streaming Boolean اختیاری. آیا ویدیو از پخش جریانی پشتیبانی می‌کند؟
