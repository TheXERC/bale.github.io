انواع کیبورد

ساختارهای داده‌ای مربوط به کیبوردهای سفارشی در بله

ReplyKeyboardMarkup

این شیء نمایندهٔ یک کیبورد سفارشی با گزینه‌های پاسخ است که به جای کیبورد معمولی نمایش داده می‌شود.

فیلد نوع توضیحات
keyboard Array of Array of KeyboardButton آرایه‌ای از ردیف‌های دکمه‌ها. هر ردیف یک آرایه از دکمه‌ها است.
resize_keyboard Boolean اختیاری. اگر true باشد، کیبورد هنگام نمایش تغییر اندازه می‌دهد.
one_time_keyboard Boolean اختیاری. اگر true باشد، کیبورد پس از اولین استفاده مخفی می‌شود.
input_field_placeholder String اختیاری. متن placeholder برای فیلد ورودی
selective Boolean اختیاری. اگر true باشد، کیبورد فقط برای کاربران خاصی نمایش داده می‌شود.

مثال:

{
  "keyboard": [
    [{"text": "گزینه ۱"}, {"text": "گزینه ۲"}],
    [{"text": "گزینه ۳"}]
  ],
  "resize_keyboard": true,
  "one_time_keyboard": false
}

KeyboardButton

این شیء نمایندهٔ یک دکمه در کیبورد سفارشی است.

فیلد نوع توضیحات
text String متن دکمه. اگر خالی باشد، دکمه نادیده گرفته می‌شود.
request_contact Boolean اختیاری. اگر true باشد، با فشار دادن دکمه، اطلاعات تماس کاربر ارسال می‌شود.
request_location Boolean اختیاری. اگر true باشد، با فشار دادن دکمه، موقعیت مکانی کاربر ارسال می‌شود.
request_poll PollType اختیاری. اگر تنظیم شود، کاربر می‌تواند نظرسنجی ایجاد کند.
web_app WebAppInfo اختیاری. اگر تنظیم شود، با فشار دادن دکمه، Mini App باز می‌شود.

InlineKeyboardMarkup

این شیء نمایندهٔ یک کیبورد اینلاین است که زیر پیام‌ها نمایش داده می‌شود.

فیلد نوع توضیحات
inline_keyboard Array of Array of InlineKeyboardButton آرایه‌ای از ردیف‌های دکمه‌های اینلاین. هر ردیف یک آرایه از دکمه‌ها است.

مثال:

{
  "inline_keyboard": [
    [
      {"text": "لینک", "url": "https://example.com"},
      {"text": "دکمه callback", "callback_data": "data123"}
    ],
    [
      {"text": "انتخاب دیگر", "callback_data": "data456"}
    ]
  ]
}

InlineKeyboardButton

این شیء نمایندهٔ یک دکمه در کیبورد اینلاین است.

فیلد نوع توضیحات
text String متن دکمه
url String اختیاری. لینک HTTP یا HTTPS که با فشار دادن دکمه باز می‌شود.
callback_data String اختیاری. داده‌ای که با فشار دادن دکمه به بازو ارسال می‌شود (حداکثر ۶۴ بایت).
switch_inline_query String اختیاری. اگر تنظیم شود، با فشار دادن دکمه، کاربر به حالت Inline Query هدایت می‌شود.
switch_inline_query_current_chat String اختیاری. مشابه switch_inline_query اما برای گفتگوی فعلی.
copy_text CopyTextButton اختیاری. اگر تنظیم شود، با فشار دادن دکمه، متن کپی می‌شود.
web_app WebAppInfo اختیاری. اگر تنظیم شود، با فشار دادن دکمه، Mini App باز می‌شود.

نکته: حداقل یکی از فیلدهای url، callback_data، switch_inline_query، switch_inline_query_current_chat، copy_text یا web_app باید تنظیم شود.

ReplyKeyboardRemove

با ارسال این شیء، کیبورد سفارشی قبلی حذف و کیبورد معمولی نمایش داده می‌شود.

فیلد نوع توضیحات
remove_keyboard Boolean همیشه باید true باشد. درخواست حذف کیبورد سفارشی را نشان می‌دهد.
selective Boolean اختیاری. اگر true باشد، حذف کیبورد فقط برای کاربران خاصی انجام می‌شود.

مثال:

{
  "remove_keyboard": true,
  "selective": false
}
