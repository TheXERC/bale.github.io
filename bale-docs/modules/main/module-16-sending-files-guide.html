راهنمای ارسال فایل‌ها

روش‌های مختلف ارسال فایل در API بله

بررسی کلی

برای ارسال فایل‌ها (عکس، ویدیو، صوت، سند و...) در بله، سه روش اصلی وجود دارد:

  1. استفاده از file_id فایل‌های قبلاً آپلود شده
  2. ارسال با استفاده از URL عمومی
  3. آپلود مستقیم فایل از طریق multipart/form-data

ارسال با استفاده از file_id

این روش سریع‌ترین و کارآمدترین روش برای ارسال فایل‌هایی است که قبلاً توسط بازوی شما آپلود شده‌اند.

مزایا:

  • سرعت بالا (بدون نیاز به آپلود مجدد)
  • مصرف کمتر پهنای باند
  • عدم محدودیت حجم (چون فایل قبلاً آپلود شده)

محدودیت‌ها:

  • فایل باید قبلاً توسط همان بازو آپلود شده باشد
  • file_id‌ها بین بازوهای مختلف متفاوت هستند

مثال:

{
  "method": "sendPhoto",
  "chat_id": 123456789,
  "photo": "AgACAgIAAx..." // file_id عکس قبلاً آپلود شده
}

نکته: file_id‌ها معمولاً طولانی هستند و با حروف بزرگ و کوچک و اعداد شروع می‌شوند.

ارسال با استفاده از URL

می‌توانید فایل را از یک URL عمومی HTTP یا HTTPS ارسال کنید.

مزایا:

  • نیاز به آپلود فایل ندارید
  • مناسب برای فایل‌های میزبانی شده در سرورهای دیگر

محدودیت‌ها:

  • URL باید عمومی و قابل دسترس باشد
  • سرور بله باید بتواند به URL دسترسی داشته باشد
  • حداکثر حجم فایل: ۲۰ مگابایت برای عکس، ۵۰ مگابایت برای سایر فایل‌ها

مثال:

{
  "method": "sendPhoto",
  "chat_id": 123456789,
  "photo": "https://example.com/images/photo.jpg"
}

نکته: URL باید مستقیماً به فایل اشاره کند، نه به یک صفحه وب.

آپلود مستقیم فایل

برای ارسال فایل‌های جدید، باید آن‌ها را به صورت multipart/form-data آپلود کنید.

مزایا:

  • امکان ارسال هر فایلی
  • کنترل کامل روی محتوای فایل

محدودیت‌ها:

  • حداکثر حجم فایل: ۱۰ مگابایت برای عکس، ۵۰ مگابایت برای سایر فایل‌ها
  • سرعت پایین‌تر به دلیل آپلود
  • مصرف بیشتر پهنای باند

روش آپلود:

  1. فایل را به عنوان بخشی از درخواست multipart/form-data ضمیمه کنید
  2. در پارامتر مربوطه، از attach://<نام_فایل> استفاده کنید
  3. فایل را با همان نام در بخش files درخواست قرار دهید

مثال با curl:

curl -X POST "https://api.bale.ai/bot<TOKEN>/sendPhoto" \
  -F "chat_id=123456789" \
  -F "photo=@/path/to/photo.jpg" \
  -F "caption=توضیحات عکس"

مثال با Python (requests):

import requests

TOKEN = "YOUR_BOT_TOKEN"
chat_id = 123456789

with open('photo.jpg', 'rb') as photo:
    files = {'photo': photo}
    data = {
        'chat_id': chat_id,
        'caption': 'توضیحات عکس'
    }
    response = requests.post(
        f'https://api.bale.ai/bot{TOKEN}/sendPhoto',
        files=files,
        data=data
    )

محدودیت‌های حجم فایل

نوع فایل حداکثر حجم
عکس (Photo) ۱۰ مگابایت
ویدیو (Video) ۵۰ مگابایت
انیمیشن (Animation/GIF) ۵۰ مگابایت
صوتی (Audio) ۵۰ مگابایت
پیام صوتی (Voice) ۵۰ مگابایت

بهترین روش‌ها

  • برای فایل‌های تکراری، همیشه از file_id استفاده کنید
  • قبل از آپلود مستقیم، بررسی کنید که فایل قبلاً آپلود نشده باشد
  • برای فایل‌های بزرگ، از URL استفاده کنید اگر ممکن است
  • همیشه caption مناسب برای فایل‌ها اضافه کنید
  • برای عکس‌ها، کیفیت مناسب را انتخاب کنید تا حجم کمتری داشته باشند
